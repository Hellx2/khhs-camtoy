CAM Toy

A detailed guide

Welcome to the CAM toy unit of KHHS technology. In this unit you will learn how to act inside of a workshop and will create a CAM toy using the skills you are taught.

Firstly, you'll need to know the terms for the topic. A "cam" is a circle of wood, which looks like this:




Secondly, you'll need to learn the correct safety protocols, so that you can use equipment safely in a workshop. The following rules should be followed at all times:
  • Never run in the workshop.
  • Wear safety goggles whenever operating a machine
  • Always wear an apron while in the workshop.
  • When using the drill or scroll saw, hold your job.
         
  • Never play with equipment.
  • Never use equipment without supervision.
  • Never use equipment you aren't supposed to.
